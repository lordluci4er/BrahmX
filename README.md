# 🤖 BrahmX

BrahmX is an intelligent AI chatbot built with NLP and Transformer models.

## 🚀 Features
- Spell correction (SymSpell)
- NLP processing (spaCy)
- Transformer summarization (HuggingFace)
- Multi-source knowledge (Wikipedia + DuckDuckGo)
- Real-time streaming responses

## 🧠 Tech Stack
- Python
- Flask
- spaCy
- SymSpell
- Transformers (HuggingFace)

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python server.py
